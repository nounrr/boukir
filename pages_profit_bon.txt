Pages/fichiers qui affichent le profit d'un bon

1. frontend/src/pages/BonsPage.tsx
   - Affiche le profit calcule pour un bon dans la liste/details des bons.
   - References principales:
     - computeMouvementDetail(...)
     - affichage: {formatNumber4(profit)} DH

2. frontend/src/components/BonFormModal.tsx
   - Affiche le profit des lignes dans le formulaire d'un bon.
   - Affiche aussi le preview du profit du mouvement.
   - References principales:
     - calcul ligne: const profit = (pv - cr) * q - remise * q
     - affichage: {profit.toFixed(2)}
     - mouvementPreviewResp.mouvement_calc.profit

3. frontend/src/pages/ChiffreAffairesDetailPage.tsx
   - Affiche le profit par bon dans le detail du chiffre d'affaires.
   - Affiche aussi le profit par ligne du bon.
   - References principales:
     - calcul.profitBon
     - item.profit
     - texte: Profit Net Bon
     - texte: % Profit Bon

4. frontend/src/pages/StatsDetailPage.tsx
   - Affiche les profits detailles par produit/client et details de bons.
   - References principales:
     - totals.totalProfit
     - row.totalProfit
     - client.profit
     - d.profit
     - product.profit

Pages qui affichent un profit global, pas directement le profit d'un bon

5. frontend/src/pages/ChiffreAffairesPage.tsx
   - Affiche les totaux de profit/chiffre beneficiaire par jour ou periode.
   - Ce n'est pas le detail d'un bon, mais les lignes peuvent ouvrir ChiffreAffairesDetailPage.

6. frontend/src/pages/DashboardPage.tsx
   - Affiche le taux de profit du jour dans le tableau de bord.
   - Ce n'est pas le detail d'un bon.

7. frontend/src/pages/ReportsPage.tsx
   - Affiche le benefice net global des rapports.
   - Ce n'est pas le detail d'un bon.

8. frontend/src/pages/ContactsPage.tsx
   - Calcule un profit pour certains elements e-commerce/avoirs.
   - Ce n'est pas l'affichage direct du profit d'un bon classique.
